# Rust Examples

This list mirrors the Rust examples wired into `openinfer/Cargo.toml`.

- `examples/openinfer/mlp_regression.rs`: Basic MLP with dynamic input.
- `examples/openinfer/residual_mlp_stack.rs`: Residual stack with repeated blocks.
- `examples/openinfer/linear_attention.rs`: Linear attention pipeline example.
- `examples/openinfer/moe_routing.rs`: Mixture-of-experts routing logic.
- `examples/openinfer/streaming_pipeline.rs`: Yield/await streaming control flow.
- `examples/openinfer/kv_cache_decode.rs`: KV cache decode step with persistent memory.
- `examples/openinfer/online_weight_update.rs`: Persistent weight update loop.
- `examples/openinfer/quantized_linear.rs`: Packed + accumulate matmul example.
- `examples/openinfer/stability_guard.rs`: Stability checks and control flow.
- `examples/openinfer/cache_window_slice.rs`: Cache table slicing behavior.
